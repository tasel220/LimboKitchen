Dialogue
Restaurant	Gion
도운 : 주문하신 음식 나왔습니다.
??? : (개운한) 하...
도운 : 음식이 입에 맞지 않으신가요?
??? : 아니. 음식은 괜찮아. 오히려 덕분에 술기운이 좀 가신 것 같군.
기온 : 늦었지만 내 소개부터 하지. 내 이름은 기온이야.
도운 : 도운이라고 합니다. 기온씨도 여행 중이신 건가요?
기온 : 음... 이걸 여행이라고 표현해도 될지 모르겠네.
기온 : 놀러 왔다기 보단 도망 왔다고 하는 편이 더 맞을 것 같은데.
도운 : (당황한) ...
기온 : (웃으며) 네가 상상하는 그런 일은 아니니까 너무 걱정하지 말라고. 보기에는 이래도 나름 모범 시민으로 살아왔으니.
도운 : 그럼 무엇으로부터 도망치신 건가요?
기온 : (기운 없는) 삶으로부터 도망친 거지. 지긋지긋한 일상으로부터.
기온 : (자조하며) ...참 한심하지? 다 큰 어른이 아직도 어린 아이 같은 소리나 하고 있으니.
도운 : 무슨 일이 있으셨는지는 모르겠지만, 너무 스스로를 몰아세울 필요는 없지 않을까요?
기온 : (기운 없는) ...네가 말동무라도 되어줄래? 이 나이가 되고 보니 어디 내 이야기 풀어놓을 곳도 마땅히 없어서 말이야.
도운 : 저는 괜찮아요. 이야기 해주시면 감사히 듣겠습니다.
기온 : (웃으며) 고맙다.
기온 : ...
기온 : (한숨) 음... 막상 말을 꺼내려니 어디서부터 시작해야 할지 모르겠군.
도운 : 천천히 생각하세요. 시간은 많아요.
기온 : (웃으며) 물론, 급할 건 없지.
기온 : 하지만 이상하게 나는 서두르지 말자고 생각할수록 더 초조해지더라고.
기온 : (기운 없는) 결국 그 조급함 때문에 모든 일을 그르치고 나서도 예전 버릇은 쉬이 고쳐지지 않더군.
도운 : 불안해서 그런 게 아닐까요?
도운 : 보통 불안하면 누구나 마음이 급해지기 마련이잖아요.
기온 : 그래. 듣고 보니 그랬던 것 같다.
기온 : 그때 나는 불안감에 사로잡혀 있었어. 이대로는 영영 별 볼 일없는 사람으로 남지 않을까하는 생각 말이야.
기온 : 내겐 같은 회사에서 근무하는 친구가 하나 있었다. 그는 사원 시절부터 수많은 동기들을 제치고 두각을 나타낼 만큼 우수한 인재였지. 반면 나는 발에 채일 만큼 흔한 월급쟁이들 중 하나일 뿐이었고.
기온 : 그의 존재는 내게 강력한 자극제가 되어주었어. 항상 그와 같이 되고 싶다고 생각했다. 여러 가지로 도움도 많이 받았고.
도운 : 좋은 친구를 두셨군요.
기온 : 곁에 있으면 마음이 편해지는 사람이었어. 경력은 그 친구가 조금 더 길었지만 워낙 오래전부터 알던 사이라 그런 점은 문제가 되지 않았다. 
기온 : (기운 없는) 하지만 분수에 맞지도 않는 내 꿈은 문제였지. 선두는 고사하고 평균을 유지하는 것도 내게는 힘겨운 일이었으니까. 뛰어난 재능을 발판으로 승승장구하는 선후배와 동기들을 볼 때마다 무력감을 느꼈어.
기온 : (기운 없는) 처음에는 능력이 조금 모자라도 열심히 노력하면 언젠가 인정받을 수 있다고 믿었다. 끈질기게 한 발자국씩 내딛는 거북이가 결국 게으른 토끼를 이긴다고.
기온 : (자조하며) 세상에는 노력하는 토끼도 얼마든지 널려있는데 말이야.
도운 : ...그래서 여기로 오신 건가요?
기온 : (한숨) 차라리 거기서 포기했더라면 여기까지는 오지 않았겠지.
기온 : 왕성했던 혈기도 나이가 들면서 사그라들고, 주위를 둘러보니 생각보다 많은 사람들이 그런대로 자기 자리에 적응하면서 살고 있었어. 나도 더 이상 아등바등하면서 살지 말고 이미 가진 것에 만족해야하나 고민했지. 
기온 : 그러던 중 내게 생각지도 못했던 기회가 찾아왔다. 친구가 나를 새 프로젝트 팀의 책임자로 추천한 거야. 회사에서도 우리에게 거는 기대가 컸던 만큼 나로서는 다시없을 기회였어.
기온 : 후회가 남지 않도록 최선을 다할 작정이었다. 잠도 제대로 자지 못하고 끼니도 대충 때우는 날이 많았지만 이번이 마지막 기회라고 생각하며 버텼어.
기온 : (한숨) 그게 실수였다. 길게 보지 못하고 눈앞의 일만 생각했던 것이.
도운 : (걱정스러운) 혹시 건강에 문제라도 생기신 건가요?
기온 : 시작은 단순한 두통이었어. 일을 시작하고부터 하루에도 몇 번씩 심한 두통이 찾아왔다. 하지만 팀 내에서도 수면 부족으로 비슷한 증상에 시달리는 사람이 많아서 나는 크게 신경 쓰지 않았어.
기온 : 한참이 지난 다음에야 나는 무언가 잘못되어가고 있다는 걸 깨달았다.
기온 : 언젠가부터 아무리 애를 써도 일이 전혀 눈에 들어오지 않게 되었어. 예전 같으면 금방 해치웠을 일도 처리하는데 배 이상의 시간이 걸렸고, 부하직원들과 말다툼을 하는 일도 점점 잦아졌다.
기온 : (기운 없는) 그리고는 어느 날 둑이 터지듯 나는 한순간에 무너져 버렸어. 더 이상 아무것도 할 수 없겠다는 생각이 들었다. 나는 일을 그만두고 도망치듯 회사를 떠났어.
도운 : (걱정스러운) ...친구분은요?
기온 : (찡그린) 아무 말도 하지 않았다. 할 수 없었어. 나처럼 무책임한 사람을 믿어준 그 친구의 얼굴을 마주볼 용기가 도저히 나지 않았다.
기온 : (찡그린) 그리고 모든 것을 망쳐버린 나를, 용서할 수 없었어.
도운 : (걱정스러운) 그래도...
도운 : 그래도 제가 친구분이라면 연락해주기를 기다릴 것 같아요.
도운 : (걱정스러운) 물론 갑자기 벌어진 일이라 당황스럽기도 하고 어쩌면 화도 낼 수 있겠지만...
도운 : 친구분은 기온씨를 걱정하고 있을 거예요.
기온 : (한숨) ...
기온 : (한숨) 만약 아니라면...? 나한테 이미 정이 다 떨어져서 꼴도 보기 싫다고 하면... 그때는 정말 어떻게 해야 할지 모르겠다...
도운 : 그렇다고 하더라도 만나는 편이 더 좋지 않을까요?
도운 : 앞으로 어떻게 될지는 몰라도, 지금까지 그분은 기온씨의 좋은 친구였잖아요. 이대로 아무것도 하지 않고 멀어진다면 그건 그동안 두 분이 쌓아온 우정에 대한 예의가 아니라고 생각해요.
도운 : 물론 그 전에 건강부터 좀 더 챙기셔야겠지만...
기온 : (한숨) ...
기온 : (자조하며) 네 말이 맞다. 이대로 있어봤자 나아지는 건 없겠지.
/ 가게 문 밖에서 누군가 기온을 부르는 소리가 들린다. /
기온 : (웃으며) 고맙다. 오늘 네게 큰 신세를 졌어.
도운 : 친구분과는 잘 해결되길 빌게요.
기온 : 그래. 너도 다시 보는 날까지 건강해라.
/ 기온이 문을 열고 나간다. /
