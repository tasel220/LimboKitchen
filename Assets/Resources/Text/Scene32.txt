Dialogue
Restaurant	Mina
도운 : 아저씨 스타일이라 마음에 드실지 모르겠네요.
미나 : 그런대로 먹을 만해요. 아저씨.
도운 : 지금 일부러 그러는 거죠?
미나 : 반쯤은 그런 것도 있고, 어떻게 불러야 될지 잘 모르겠어서 그런 것도 있어요. 누구 씨 하는 건 너무 어색하지 않아요? 아직 전 고등학생인데.
도운 : 듣고 보니 그렇군요.
미나 : 그러니까 이해하세요. 어차피 아저씨라고 부른다고 나이 몇 살 더 먹는 것도 아니니까.
도운 : 되게 시니컬하시네요.
미나 : 허구한 날 침대에만 누워있어 보세요. 그게 비결이죠.
도운 : (걱정스러운) 어디 몸이라도 안 좋은 건가요?
미나 : (짜증난) 아, 제발. 그렇게 걱정하는 눈으로 바라보지 말아줄래요? 그런 시선은 질리도록 받아왔으니까. 
미나 : (짜증난) 왜 다들 이런 이야기만 나오면 몸 둘 바를 몰라 하면서 갑자기 측은한 눈빛으로 절 바라보는 거죠? 그러니까 제가 정말 불행한 사람 같잖아요.
미나 : (짜증난) 저도 즐거운 일이 있으면 웃기도 하고, 상태가 괜찮을 때는 밖에 나가서 돌아다니기도 해요. 솔직히 몸이 좀 아픈 건 사실이지만 누구나 고민 한 가지 정도는 안고 살지 않나요?
미나 : (한풀 꺾인) 물론 제 경우에는 그렇게 생각한다는 거지만...
도운 : 미나 학생 생각이 그렇다면 적어도 본인 인생에 있어서는 맞는 말이죠. 어쨌든 삶에 의미를 부여하는 건 전적으로 스스로에게 달려있으니까요.
미나 : 오글거리는 말을 아무렇지도 않게 하시네요.
도운 : 이거 나름 유명한 말이지 않나요? 어디서 들었는지 기억은 안 나지만.
미나 : 저는 그런 거 별로 안 좋아해요. 사춘기 소년 감성 같아.
도운 : 혹시 평소에 감정이 메말랐다는 말 못 들어봤어요?
미나 : 전혀요. 오히려 전 스스로 감성이 되게 풍부하고 생각하는데요. 그것 때문에 여기 올 때도 부모님께 한소리 들었는걸요.
도운 : 무슨 일 때문에요?
미나 : 계속 교복을 입겠다고 고집을 부렸거든요. 입학하고 나서 실제로 등교를 해본 적이 거의 없어서 고등학생이 된 기분 좀 내보고 싶었어요.
미나 : 원래대로라면 저는 초등학교도 졸업하지 못할 예정이었어요. 의사 선생님들은 12살 정도가 한계일 거라고 말씀하셨죠.
미나 : 그래도 저희 엄마는 중학교 교복까지 미리 준비했었대요. 그래야 제가 더 오래 살 것 같았다고.
미나 : 덕분인지 모르겠지만 지금까지 큰 탈 없이 살고 있으니 엄마 소원은 이루어진 셈이죠.
도운 : 그럼 여기는 요양하러 온 거예요?
미나 : 그렇다고 하는데, 솔직히 저는 이 정도로 깊은 산골일 거라고는 생각 못했어요. 요즘 같은 세상에 아직도 전화가 안 터지는 곳이 있다니.
미나 : 덥지 않아서 좋긴 한데 분위기도 왠지 으스스하고.
도운 : 무슨 말인지 알 것 같네요. 사실 저도 안개 때문에 길을 잃어서 여기까지 왔거든요. 처음에 여기 사장님하고 만났을 때는 귀신이라도 나온 줄 알았다니까요.
미나 : 아저씨는 귀신이 실제로 있다고 믿어요?
도운 : 진지하게 믿는 건 아니지만 있을지도 모른다고는 생각해요.
도운 : 미나 학생은요?
미나 : 모르겠어요. 그리고 만약 죽음 이후에 뭔가 있다고 해도 그게 좋은 일인지는 장담할 수 없잖아요. 겪어보기 전에는 모르는 일이죠.
미나 : (웃으며) 만약 알게 되면 내가 아저씨한테 알려주러 올게요. 저 세상은 어떤 곳인지.
도운 : (당황한) ...
미나 : 뭘 그렇게 놀라고 그래요? 설마 무서워서 그런 건 아니죠?
도운 : (당황한) 아니요. 그건 아닌데...
도운 : 너무 아무렇지도 않게 그런 이야기를 해서 조금 당황했어요.
미나 : 아무렇지도 않은 건 아니에요. 말은 그렇게 해도 저는 누구보다 삶을 사랑하는 사람이라구요. 
미나 : 그저 익숙해졌을 뿐이에요. 인간은 누구나 죽는다는 사실을 모르는 사람은 없지만 보통은 먼 훗날의 일이라고 생각하잖아요.
미나 : 반면에 저는 매일 아침 눈을 뜰 때마다 생각해요. ‘오늘도 살았구나.’
미나 : 따지고 보면 정말 놀랍지 않나요? 한 치의 의심도 없이 내일도 눈을 뜰 수 있을 거라고 믿으면서 잠자리에 드는 사람들 말이에요.
미나 : 어렸을 때 저는 그런 종류의 두려움에 빠진 적이 있었어요. 이대로 영원히 잠들어서 다시는 일어나지 못하면 어쩌나 진지하게 고민했었죠.
도운 : 그럼 잠을 잘 수가 없잖아요.
미나 : 맞아요. 그것 때문에 한동안 뜬눈으로 밤을 새웠어요. 부모님이 많이 걱정하셨죠. 절대 네가 생각하는 그런 일은 일어나지 않을 거라고 몇 번이나 말씀하셨어요. 하지만 그래도 저는 안심이 되지 않았죠.
미나 : 결국 제가 눈을 뜨지 않으면 다시 일어날 때까지 옆에서 계속 깨워주기로 부모님께 약속을 받고 나서야 겨우 잠이 들 수 있었어요.
미나 : (웃으며) 지금 생각해보니 정말 손이 많이 가는 아이였네요.
도운 : 혹시 요즘도 그런 건 아니죠?
미나 : (빨개진) 이건 어릴 때 이야기라니까요!
도운 : (웃으며) 농담이에요.
도운 : 그럼 이제 어디로...
/ 갑자기 시야가 흐려진다. /
도운 : (당황한) 어...?
미나 : (당황한) 아저씨? 왜 그래요?
/ 시야가 어두워지기 시작한다. /
